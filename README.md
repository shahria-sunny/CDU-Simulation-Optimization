# Crude Oil Atmospheric Distillation: Simulation and Energy Optimization
This repository contains a steady-state simulation and a comprehensive technical report for an **Atmospheric Crude Distillation Unit (CDU)** processing **Arabian Light Crude Oil**. 
## 🚀 Project Overview
The primary objective of this study was to achieve significant energy savings through structured heat integration and furnace duty optimization.
* **Software:** Aspen HYSYS V14
* **Fluid Package:** Peng-Robinson Equation of State (PR-EOS)
* **Feedstock:** Arabian Light Crude (33.4° API)
* **Optimization Result:** **53.6% reduction** in furnace duty (approx. 81.3 MW saved)
## 🛠 Technical Implementation
* **Feed Characterization:** 30 pseudo-components generated from TBP assay data.
* **Column Configuration:** 35-stage main fractionator with three side draws (Naphtha, Kerosene, Diesel).
* **Heat Integration:** Implementation of two pump-around circuits to optimize internal heat distribution and recover thermal energy for crude preheating.
* **Thermodynamic Analysis:** Supported by conceptual Pinch Analysis and Exergy Assessment to identify high-priority heat sources (Atmospheric Residue).
